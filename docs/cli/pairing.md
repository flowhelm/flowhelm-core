# `flowhelm pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
flowhelm pairing list whatsapp
flowhelm pairing approve whatsapp <code> --notify
```
