# `flowhelm tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
flowhelm tui
flowhelm tui --url ws://127.0.0.1:18789 --token <token>
flowhelm tui --session main --deliver
```
